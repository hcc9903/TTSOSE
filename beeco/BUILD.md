# BeeCo

这是一个简化版的 Flutter 记账应用，从 BeeCount 项目重构而来。

## 快速开始

1. 确保已安装 Flutter SDK (>=3.0.0)
2. 进入项目目录: `cd beeco`
3. 安装依赖: `flutter pub get`
4. 运行应用: `flutter run`

## 构建 APK

```bash
flutter build apk --release
```

APK 文件将生成在 `build/app/outputs/flutter-apk/app-release.apk`

## 功能

- 多账本管理
- 收支记录
- 分类管理
- 账户管理
- 数据统计
- 暗黑模式
- 数据导入导出

## 注意事项

这是一个简化版本，移除了以下复杂功能：
- 云同步
- AI 功能
- 截图监听
- 桌面小组件
- 内购功能
